Exports types for [tcomb-form](https://github.com/gcanti/tcomb-form).

## Installation
```
npm install --save tcomb-form-types
```

## Usage
```js
import t from 'tcomb-form';
import { email } from 'tcomb-form-types';

const test = t.struct({
  email: email
});
```
